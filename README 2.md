# Instrucciones para ejecutar la aplicación Angular

## Pre Requisitos

* Angular CLI v6
* Node.js v8+ 

Ingrese a la carpeta del proyecto Angular

```
cd code-team-client
```

Instale las dependencias

```
npm install
```

Levantar la aplicación usando el siguiente comando:

```
ng serve --aot
```
